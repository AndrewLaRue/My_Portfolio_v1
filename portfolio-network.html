Network

VueJs

Welcome to The Network! The CodeWorks Alumni Network is an online platform used by CodeWorks students to stay connected as they continue their journey as software developers!
In this application, I created a client interface that utilizes a social networking API. Network is a SPA using the VueJS framework and making use of Vue-Router in order to create searches to the API utilizing query parameters, while working with Auth0 to manage client identity.
I was responsible for building the client side of the application in accordance with an existing API
